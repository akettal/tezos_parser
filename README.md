# Tezos parser

A typescript tool for decoding Tezos RPC response for smart contracts storage into usable json object


## Running the example

```
npm install

npm run example
```

## Data type supported

- [x] Address
- [x] String
- [x] Big Map
- [x] Map
- [x] Bool
- [x] Nat
- [x] Mutez
- [x] Or
- [x] Pair
- [x] Left
- [x] Right
- [ ] Contract
- [ ] List
